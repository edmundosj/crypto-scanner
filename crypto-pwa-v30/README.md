# 📱 Crypto Pattern Scanner — PWA

Monitor de padrões técnicos de criptomoedas com suporte a Progressive Web App.

## 📁 Arquivos

```
crypto-pwa/
├── index.html      ← App completo (HTML + JS + CSS)
├── manifest.json   ← Configuração PWA
├── sw.js           ← Service Worker (cache offline)
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## 🚀 Como hospedar (grátis)

### Opção 1: Netlify Drop (mais fácil)
1. Acesse https://app.netlify.com/drop
2. Arraste a pasta `crypto-pwa` inteira para a página
3. Pronto! Você recebe um URL `https://xxxx.netlify.app`

### Opção 2: GitHub Pages
1. Crie um repositório no GitHub
2. Faça upload dos arquivos
3. Vá em Settings → Pages → selecione a branch main
4. URL: `https://seu-usuario.github.io/nome-do-repo`

### Opção 3: Vercel
1. Instale o Vercel CLI: `npm i -g vercel`
2. Dentro da pasta: `vercel --prod`

## 📲 Instalar no celular

### Android (Chrome)
- Abra o URL no Chrome
- Toque nos 3 pontinhos → "Adicionar à tela inicial"
- Ou aguarde o banner de instalação aparecer automaticamente

### iPhone (Safari)
- Abra o URL no Safari
- Toque no ícone de compartilhar (⬆️)
- "Adicionar à Tela de Início"

## ⚡ Recursos PWA

- ✅ Instalável como app nativo
- ✅ Ícone na tela inicial
- ✅ Abre em tela cheia (sem barra do browser)
- ✅ Cache offline para assets estáticos
- ✅ Banner de instalação automático
- ✅ Indicador de conexão offline
- ✅ Navigation bar mobile
